# Soul Color: The Judgment

An H5 personality test game with 40 character archetypes.

## Features
- 40 character archetypes with unique animations
- 800+ psychological test questions
- Random question selection
- Anime character images
- Background music (Sakura Tears)
- Mobile responsive design

## How to Play
1. Open index.html in a web browser
2. Answer 40 questions
3. Get your character type result

## Project Structure
- index.html - Main game file
- style.css - Styles
- script.js - Game logic
- music-player.js - Music player
- images/ - Character images
- Sakura Tears.mp3 - Background music
